# Teacher Test Maker

教師がテストを簡単に作成・印刷・エクスポートできるWebアプリです。

---

## 🌍 使い方（これだけ）

1. 下のURLを開く
   https://fanyesena-pixel.github.io/teacher-test-maker/

2. 問題を入力する

3. 「生徒用PDF保存・印刷」を押す

4. 印刷またはPDFとして保存

---

## ✨ 主な機能

* テスト作成
* 生徒用プリント生成
* 解答付きプリント生成
* CSVエクスポート
* 問題バンク保存

---

## 💡 特徴

* インストール不要
* ブラウザだけで使える
* 完全無料

---

## ⚠️ 注意

* 作成データはブラウザに保存されます
* 別のPCでは引き継がれません
* 必要ならJSONバックアップを使ってください
