先生用テスト作成室

問題を手入力して、中央の紙プレビューを見ながらテストを作れる学校向けのWebアプリです。 生徒用PDF、解答付きPDF、解答用紙、CSV、バックアップJSONまでを一つの画面で扱えます。 第2段階では問題バンク、A/B/Cパターン、配点補助、テンプレート機能も追加しました。

Guide

使い方の流れ

入口から本体へ移動

`ページ本体へ進む` から編集画面へ移動します。LPは説明用、`app.html` 側は作業用に分かれています。

中央の紙を見ながら作成

左でテンプレートや配点、右で問題編集と問題バンク、中央で生徒用・解答付き・解答用紙を確認できます。

学校で使える形で保存

生徒用PDF、解答付きPDF、解答用紙、CSV、バックアップJSONを用途に応じて出力できます。

Features

今回入っている主な機能

問題バンク

作った問題を保存し、次のテストへすぐ追加できます。お気に入り、タグ絞り込み、難しさ絞り込みにも対応しています。

A/B/Cパターン

元の問題は残したまま、問題順シャッフルや選択肢シャッフルを使ったA/B/Cパターンを作成できます。

配点補助

100点化、50点化、全問同配点、記述だけ高配点などをワンクリックで行い、合計点も自動で再計算します。

Start

準備ができたら、編集画面でそのままテストを作れます。

説明はこのLPに残し、作業は編集画面に集中できるように分けています。
